1. 2D платформер
2. Пользовательская кастомизация оружия за валюту (по возможности)
3. Базовая идея - персонаж двигается по карте (вид камеры сбоку) и сражается с мобами
4. Пользовательская кастомизация персонажа за некоторую валюту
5. Разные виды оружия (пистолеты, мечи, луки, гранаты...) доступные за некоторую валюту (или выпадающие с боссов)
6. Дерево навыков для персонажа (ограниченное), дерево прокачки оружия. Базовые навыки - движение, прыжки, кувырки, удары руками/ногами 
7. Фичи для навыков: замедление времени, гиперпрыжок, гиперрывок, невосприимчивость к ударам, крюк, повышение выносливости, повышение здоровья, уворот от пуль, телепорт...
8. Каждые k уровней происходит смена локации: локации - планеты, перемещение между планетами - на ракете, перемещение внутри локации перемещение на лифте/вертолете...
9. Валюта начисляется за скорость прохождения уровня и количество полученного урона...
10. На карте возможны лазеры и другие препятствия
11. Для перехода на новую планету нужно пройти босса 
12. Меню для навигации по уровням (перепройти предыдущие либо пройти следующий). Чем сложнее уровень, тем больше валюты получает игрок.
13. Разрушение некоторых элементов локации для продвижения дальше по уровню (например, двери/стены)
14. Элементы головоломок для дальнейшего продвижения внутри локации
15. Инвентарь с оружием (переключение прокруткой + дополнительная кнопка для всего инвентаря)
16. Дополнительные миниигры в промежутках между планетами(на корабле).
17. Гравитация на разных планетах разная
18. Разные фичи для разных планет(например планета водная)
19. Попытаться написать свою музыку
20. Сделать разрушаемость обьектов
21. Добавить всякие секретные комнаты
22. Добавить бусты на уровни(например временная суперсила или суперскорость)
23. Добавить портальную пушку
24. Добавить компанионов или npc-торговцев.
